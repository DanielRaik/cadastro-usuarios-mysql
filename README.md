# Cadastro de Usuários – Projeto Fullstack

Projeto pessoal de estudo desenvolvido para praticar a integração entre
Frontend, Backend e Banco de Dados.

## Tecnologias utilizadas
- HTML
- TailwindCSS
- JavaScript
- Node.js
- Cors
- Express
- MySQL

## Funcionalidades
- Cadastro de usuários
- Validação básica de campos
- Armazenamento em banco de dados MySQL
- Comunicação via API REST

## Como rodar o projeto localmente

### Backend
```bash
cd backend
npm install
node index.js
